◆ evaluate()

virtual void wassail::data::common::evaluate ( bool  force = false)
pure virtual

Evaluate the data source. The implementation should cache the result and return the cached data in subsequent calls rather than re-evaluating the data source. The exception is when force is true, then the information should be collected again.

Parameters
[in]forceFlag to specify whether the data should be collected even if it already has been evaluated.
